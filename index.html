Web Dev Cheat Sheet

Here is a collection of things to refer back to when developing in html and css.

Colours

Colours can be represented in lots of ways in css. Some provide more capabilities than others.

Name - There are a set of colours that can be referenced with a plain english name. Some examples include blue, red, blueviolet

HexCode - Colours can also be represented by a hexadecimal code. These should be written with a hashtag before like these examples. #CB3EBD, #71CE90, #9B2020

RGBA - a great method for representing colours in css that also allows you to control transparency is by using rgba. The 3 values describe the quantity of red, green and blue. Each of the values have a max of 255 except for the last value (alpha). Alpha is entered as a percentage and controlls the opacity of the text. One example of a colour represented with rgba is rgba(255, 99, 71, 0.6)
